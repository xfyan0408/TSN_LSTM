## ChannelMix 和 TCN 的分工是

| 模块       | 输入     | 输出         | 作用                               |
| ---------- | -------- | ------------ | ---------------------------------- |
| 原始输入   | T×3     | T×3         | 历史带宽、CPU、内存需求            |
| ChannelMix | T×3T    | T×D         | 每个时刻融合三类资源，提升特征维度 |
| TCN        | T×D     | T×D′       | 沿时间轴学习历史变化和滞后关系     |
| LSTM       | T×D′   | 一个窗口表示 | 汇总长期趋势                       |
| Heads      | 窗口表示 | H×3        | 输出未来 H 步三类资源需求          |
